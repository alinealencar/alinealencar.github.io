  Capstone

For my capstone, I built a web application for managing events called Kikakuya. I lead a team of 4 developers (myself included). I acted primarily as the Back-End developer and Database Administrator, but helped in the Front-End as needed. I was also responsible for the deployment of the application.

The product was built using HTML5, CSS3, Javasript, jQuery and Bootstrap for the Front-End and Java, with Spring MVC, for the Back-End. The application is hosted on AWS, running on a Tomcat 8 web server.

You can use Kikakuya at kikakuya.ca.
